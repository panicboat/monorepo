# Language Rules

## Output Language

すべての出力は**日本語**で行うこと。

## Documentation Format

ドキュメント（proposal.md、design.md、spec.md など）:

| 要素 | 言語 | 例 |
|------|------|-----|
| 見出し | 英語 | `## Implementation Details` |
| 本文 | 日本語 | 実装の詳細について説明します。 |

## Code Language

コード内の要素は**英語**で記述:

- 変数名・関数名
- コメント
- コミットメッセージ
