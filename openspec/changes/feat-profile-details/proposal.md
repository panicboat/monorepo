# Profile Details Enhancement

## Background
現在、プロフィール編集画面はオンボーディングに必要な最小限の情報（基本情報、写真、ソーシャルリンク）のみをサポートしています。
キャストの魅力を高め、ゲストにマッチングのための十分な情報を提供するために、オンボーディングでは省略された詳細なプロフィール情報（身体情報やタグなど）を入力できるようにする必要があります。

## Goal
プロフィール編集画面 (`/manage/profile`) を拡張し、詳細なプロフィール属性をサポートする。

## Changes
### Data Model (`ProfileFormData`)
データモデルを拡張し、以下を含めます：
- **Physical (身体情報)**: 年齢、身長、血液型、スリーサイズ (B/W/H + カップ)。
- **Attributes (属性)**: タグ (例: #英語OK, #コスプレ)。
- **Appeal (アピール)**: 詳細な自己紹介（現在は `bio` のみですが、一言と詳細を分けるなど）。

### UI Components
`src/modules/portfolio/components/cast/` に新しい再利用可能な入力コンポーネントを作成します：
1. **`PhysicalInputs.tsx`**:
   - 身長 (cm)、年齢 (歳)、血液型 (選択) の入力。
   - スリーサイズ (B/W/H) とカップサイズ (選択) の入力。
2. **`TagSelector.tsx`**:
   - プリセットリストからの選択、または新規タグの追加。
   - 選択されたタグをチップとして表示。

### Page Implementation
- `ProfileEditPage` を更新し、新しいセクション（「身体情報」、「アピール」）にこれらのコンポーネントを組み込みます。
- API型とMSWハンドラを更新し、これらのデータを保存できるようにします。
