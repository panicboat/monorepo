# Cast Navigation Bar Requirements

## ADDED Requirements

### Requirement: Navigation Items Update
ナビゲーションバーの項目は以下の順序と構成に変更されなければならない (MUST)。
- Home
- Schedule
- Concierge
- MyPage

#### Scenario: Verify Navigation Items
1. キャストがアプリにログインする。
2. 画面下部のナビゲーションバーを確認する。
3. 左から順に "Home", "Schedule", "Concierge", "MyPage" が表示されていることを確認する。
4. "Plan" タブが存在しないことを確認する。

#### Scenario: MyPage Navigation
1. "MyPage" タブをタップする。
2. マイページ画面 (`/manage/mypage`) に遷移することを確認する。
