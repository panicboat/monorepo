# Refactor Cast Screen Layout

## Goal
キャストの画面構成を整理し、ナビゲーションとマイページを刷新する。

## Features
- **Navigation**: "Plan" を削除し "MyPage" を追加。
- **Home**: ダッシュボードのURLを `/manage/home` に変更。
- **MyPage**: プロフィール、統計、管理項目一覧を表示する新ページ。

## Background
- 現状の "Plan" タブは頻度が低いため MyPage 内に移動。
- キャストが自分の情報を一元管理できる MyPage が必要。
