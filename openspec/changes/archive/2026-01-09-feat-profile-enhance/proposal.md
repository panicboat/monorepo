# Profile UI Enhancement

## Background
プロフィール編集ページ（`/manage/profile`）に多くの項目が追加されたため、全体の長さが伸び、スクロール量が増えています。
ユーザービリティを向上させるために、情報のグループ化や表示の整理を行う必要があります。

## Goal
- プロフィール編集画面のUIを整理し、視認性と操作性を向上させる。
- 長いスクロールを避け、必要な情報にアクセスしやすくする。

## Changes
### UI Improvement
- **カード化**: 各セクション（基本情報、写真、詳細など）をカードで区切り、視覚的なまとまりを作る。
- **アコーディオン/タブ**: 使用頻度の低い項目や詳細な項目を折りたたみ可能にするか、タブで切り替えられるようにする。
- **スティッキーヘッダー/ナビ**: ページ内リンクや保存ボタンへのアクセスを改善する（既存のフッターバーとの整合性を考慮）。

### UX
- 入力項目のグルーピングを見直し、論理的な順序にする。
